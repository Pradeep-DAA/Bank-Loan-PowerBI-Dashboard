# Bank Loan Analysis Dashboard (Power BI + SQL)

## Project Overview

This project analyzes bank loan data to monitor lending performance, loan quality, and borrower behavior.
The dashboard helps identify trends, measure loan risk, and evaluate lending efficiency.


## Business Problem

Banks need to monitor loan performance, track repayment behavior, and identify risky loans.
This dashboard provides insights into:

* Total Loan Applications
* Funded Amount
* Loan Repayment Status
* Good vs Bad Loans
* Loan Trends by Month and Region


## Tools & Technologies Used

* Power BI
* SQL Server
* Excel / CSV
* DAX
* Data Visualization


## Dashboard Pages

## Summary Dashboard

Provides high-level KPIs and loan quality insights.

Key Metrics:

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average DTI
* Good Loan vs Bad Loan %


### Overview Dashboard

Shows loan trends and distributions.

Charts Included:

* Monthly Loan Applications (Line Chart)
* State-wise Loan Distribution (Map)
* Loan Purpose Analysis (Bar Chart)
* Home Ownership Analysis (Tree Map)
* Employee Length Analysis


### Details Dashboard

Displays loan-level detailed records.

Features:

* Loan ID-level data
* Filters by State
* Filters by Grade
* Filters by Loan Status


## Files Included

## Key Insights

* Majority of loans are categorized as Good Loans (~90%)
* Debt Consolidation is the most common loan purpose
* Loan applications increase steadily over months
* Most loans are issued under Mortgage and Rent categories
